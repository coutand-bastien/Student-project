utils.classifier.vectorization_functions.vectorized_payload(payload: str)

The function vectorize_payload vectorized a payload. The vectorization process is based on the sum of the ASCII value of each character of the payload. The payload is split into 4 subsets. If the payload is less than 8 characters, the payload is split into 1 subset. :param payload: a payload :return: a vectorized payload of size 1x4 :rtype: list

>>> print(vectorized_payload("Hello"))
[448, 0, 0, 0]
